# 📸 Imágenes para el Simulador de Examen B-II B

## Instrucciones

Las preguntas del balotario que incluyen señales de tránsito necesitan sus imágenes correspondientes.
Cada imagen debe guardarse en esta carpeta con el nombre exacto indicado abajo.

### Formato aceptado
- **PNG** (recomendado), JPG o WEBP
- Tamaño recomendado: **200x200 px** a **400x400 px**
- Si usas otro formato que no sea PNG, actualiza la extensión en `preguntas.js`

- Captúralas directamente del PDF del balotario
